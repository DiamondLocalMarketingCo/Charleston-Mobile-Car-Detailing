# Charleston Mobile Car Detailing

SEO-focused mobile car detailing website built with Next.js.

## Current scope

This is the public website only. It does not include the booking tool, CRM, payments, Supabase, or SaaS functionality yet.

## Pages included

- `/` homepage
- `/services`
- `/interior-car-detailing`
- `/exterior-car-detailing`
- `/full-car-detailing`
- `/maintenance-detailing`
- `/gallery`
- `/reviews`
- `/service-areas`
- `/contact`
- `/book`
- City SEO pages for Charleston, South Charleston, Dunbar, Nitro, St. Albans, Cross Lanes, Teays Valley, and Hurricane

## Image placeholders

The design uses CSS placeholders right now. Add real images later in `/public/images/`, especially:

- `/public/images/hero-car-charleston.jpg`
- service photos
- before/after gallery photos
- review/customer result photos

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
